# Ethics Final Project: CMSC 25910

## To Run

### Flask Server Setup
1. cd into 'flask-server'
2. run 'source myenv/bin/activate'
3. run 'python server.py'

### React App Setup
1. cd into 'react/client'
2. run 'npm start'
